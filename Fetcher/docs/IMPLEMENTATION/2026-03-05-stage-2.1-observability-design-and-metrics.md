## 2. Технические детали

### 2.1. Метрики

Файл: `fetcher/metrics.py`

Объявлены следующие метрики (Prometheus):

- Pipeline metrics:
  - `fetcher_videos_downloaded_total{platform}` — Counter;
  - `fetcher_videos_failed_total{platform,reason}` — Counter;
  - `fetcher_cache_hits_total{platform}` / `fetcher_cache_miss_total{platform}` — Counters;
  - `fetcher_download_latency_seconds{platform}` — Histogram;
  - `fetcher_metadata_latency_seconds{platform}` — Histogram;
  - `fetcher_comments_latency_seconds{platform}` — Histogram.
- Platform error metrics:
  - `fetcher_youtube_429_total{operation}` — Counter;
  - `fetcher_youtube_403_total{operation,error_code}` — Counter.

Семантика и рекомендованные buckets описаны в `FETCHER_OBSERVABILITY.md`.

### 2.2. Документация

Файл: `Fetcher/docs/FETCHER_OBSERVABILITY.md`:

- фиксирует:
  - названия метрик;
  - их смысл и лейблы;
  - derived KPI (cache hit ratio, ingestion throughput);
  - основные блоки Grafana‑дашборда.
- описывает роль:
  - логов (`fetch_logs` + stdout JSON‑логи);
  - событий (`FetcherEvent`) в контексте observability.

---

## 3. Тестирование

На этом этапе:

- unit‑тесты не добавлялись (метрики представляют собой декларативные объекты Prometheus);
- логическая проверка:
  - имена/лейблы метрик согласованы с чеклистом Phase 2;
  - метрики не пересекаются по смыслу с уже существующими метриками Backend/DataProcessor.

В дальнейшем:

- потребуется интеграция метрик в воркеры и адаптеры:
  - инкременты `*_total` в случаях успеха/ошибки;
  - оборачивание ключевых операций в таймеры `*_latency_seconds`.

---

## 4. Известные ограничения и TODO

- **Метрики пока не используются в коде воркеров**:
  - нет вызовов `inc()/observe()` в `metadata/video/comments` worker’ах и адаптерах;
  - нужно будет аккуратно расставить хуки с учётом retry/resume логики.
- **Нет экспозиции метрик через HTTP endpoint**:
  - требуется либо:
    - отдельный процесс/веб‑сервер для Fetcher (аналог API DataProcessor);
    - либо интеграция метрик в существующий сервис, который запускает воркеры.
- **Grafana‑дашборд не создан**:
  - описан только целевой набор панелей, без JSON‑конфига.
