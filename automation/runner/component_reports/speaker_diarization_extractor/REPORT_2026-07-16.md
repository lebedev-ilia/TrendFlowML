## Лог ошибок и особенности

1. **ap_venv сломан на поде** (python3.14 пути → локальная машина). Исправление: использовать `/workspace/venv` (python3.11) + pip install pyannote.audio>=3.1.
2. **torchvision конфликт**: pyannote 4.0.7 ставит torch 2.13.0 → конфликт с system torchvision. Решение: `pip install torchvision` в venv.
3. **Модель не в bundled_models** на Network Volume (папки пустые). Решение: scp из локальной машины (33MB).
4. **Wall time**: ~160с на 149с аудио (1x realtime на GPU), ~155с на 28с аудио (5.5x — модель загружается заново каждый раз).
