## Технические детали

### HTTP endpoint `/metrics`

Модуль `fetcher/api.py` предоставляет:

- **FastAPI приложение** (`app`):
  - `GET /metrics` — экспорт Prometheus-метрик в формате text/plain
  - `GET /` — root endpoint с информацией о сервисе

- **Обработка ошибок**:
  - При ошибке генерации метрик возвращается пустая метрика с комментарием `# Error generating metrics\n`
  - Ошибки логируются, но не прерывают работу endpoint'а

- **Content-Type**:
  - `text/plain; version=0.0.4; charset=utf-8` (стандартный для Prometheus)

### Запуск сервера

Для запуска HTTP сервера можно использовать:

```bash
uvicorn fetcher.api:app --host 0.0.0.0 --port 8000
```

Или через Python:

```python
from fetcher.api import app
import uvicorn

uvicorn.run(app, host="0.0.0.0", port=8000)
```

### Prometheus scraping

Prometheus должен быть настроен на scraping endpoint `/metrics`:

```yaml
scrape_configs:
  - job_name: 'fetcher'
    static_configs:
      - targets: ['fetcher:8000']
    metrics_path: '/metrics'
    scrape_interval: 15s
```

### Grafana dashboard

Документ `GRAFANA_DASHBOARD.md` описывает:

1. **6 разделов dashboard** с детальными панелями
2. **PromQL запросы** для каждой метрики
3. **Рекомендуемые алерты** (Critical и Warning)
4. **Переменные** для фильтрации по платформам и временным диапазонам
5. **Пример конфигурации Prometheus** для scraping

**Основные панели**:
- Ingestion throughput (videos/minute, videos/hour, videos/day)
- Success vs Failed ratio
- Cache hit ratio
- Download latency distribution (P50, P95, P99)
- YouTube 429/403 errors
- Error analysis by reason

## Известные ограничения

- **Queue metrics**: Панели "Queue & Worker Health" пока не актуальны, так как очередь (Celery/Redis) ещё не интегрирована. После Phase 4 нужно будет добавить метрики `fetcher_queue_length` и `fetcher_active_workers`.
- **Proxy metrics**: Метрики `proxy_failure_rate` будут добавлены после реализации proxy rotation (Phase 3).
- **Circuit breaker metrics**: Метрики `circuit_breaker_tripped_total` будут добавлены после реализации circuit breaker (Phase 3).
- **Dashboard JSON**: Dashboard описан текстом, но не экспортирован как JSON. Можно создать вручную в Grafana UI или экспортировать позже.
