# Документация изменений DataProcessor API

Эта директория содержит документацию всех изменений, добавлений и обновлений в модуле API.

## Структура документации

- **CHANGELOG.md** - общий журнал изменений (хронологический)
- **IMPLEMENTATION/** - детальные отчеты о реализации этапов
- **API_CHANGES/** - документы об изменениях в API (endpoints, schemas, etc.)
- **ARCHITECTURE/** - документы об архитектурных изменениях
- **DEPRECATIONS.md** - список устаревших функций и их замена

## Формат документов

Каждый документ должен содержать:
- **Дата** изменения
- **Версия** API (если применимо)
- **Тип изменения**: добавление, изменение, удаление, исправление
- **Описание** изменений
- **Ссылки** на связанные задачи/PR/коммиты
- **Миграция** (если требуется)

## Индексация

Все документы индексируются в соответствующих индексных файлах для удобного поиска.
