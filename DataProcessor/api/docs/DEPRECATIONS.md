# Устаревшие функции

Этот документ содержит список функций, которые устарели или будут удалены в будущих версиях API.

## Формат

Каждая запись должна содержать:
- **Функция** - что устарело
- **Версия устаревания** - когда было объявлено устаревание
- **Версия удаления** - когда будет удалено (если известно)
- **Замена** - чем заменить
- **Причина** - почему устарело
- **Миграция** - как мигрировать

## Пример формата

### Endpoint: POST /api/v1/process (старая версия)

- **Версия устаревания**: 0.2.0
- **Версия удаления**: 0.3.0
- **Замена**: POST /api/v2/process
- **Причина**: Изменение формата запроса для поддержки новых функций
- **Миграция**: 
  ```python
  # Старый формат
  {
    "run_id": "...",
    "video_path": "..."
  }
  
  # Новый формат
  {
    "run_id": "...",
    "video_path": "...",
    "profile_config": {...}
  }
  ```
