## 4. Сверка с кодом

1. **`run_segments`**: strict `N`; предобработка + батч‑инференс; строки без успеха остаются NaN и `mask=false`.
2. **`segments_count`**: `int(np.sum(seg_mask))` — число **валидных** сегментов, не длина списка Segmenter при частичном фейле.
3. **Савер**: tabular через общий `add` → `as_float`; эмбеддинги без строковых полей в float‑векторе.
4. **`run()`**: отключён в audited режиме.

---

## 5. Вердикт

**Плюсы:** обязательный Baseline‑аудио компонент; полный 512‑D агрегат + per‑окно матрица; строгая временная ось и маска; offline веса; на **A** чистые float без артефактных NaN в tabular.

**Минусы:** L1 без **B/C**; лёгкое расхождение **device** в meta vs `models_used`; документация ранее расходилась с порядком tabular — выровнено.
