## Замечания (не блокируют)

1. `artifact_written` гейтируется в NaN вместе с extra-ключами при дефолте — downstream не может читать этот флаг без включения emit_extra_metrics. Задокументировано в CRITERIA.md C4.
2. Все ok-NPZ в storage от тестового Fetcher имеют одинаковые 3 комментария — различимость в реальных данных не верифицирована (синтетика достаточна).
3. GPU-прогон не делался — sentence-transformers работает на CPU, этого достаточно для валидации.
