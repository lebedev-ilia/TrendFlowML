## Логика и код

- Чанкинг token-aware (shared_tokenizer_v1); overlap_ratio=0.15; max_chunk_tokens_model=256
- L2-нормализация in `_encode_chunks` (norms[norms==0]=1 → safe div)
- Empty path: нет текста → немедленный return без модели, features_flat из `_build_features_flat({})` — 16 ключей
- Cache: отключён по умолчанию (cache_enabled=False)
- Batch: `extract_batch()` реализован, порядок чанков сохраняется через `global_chunk_idx`
- `requires_any_source`/`require_asr` — fail-fast flags (не включены по умолчанию)

## Замечания
1. `conf_mean/min/max` = NaN при `conf_present=0` — по-настоящему expected behaviour; реальный Whisper в AudioProcessor не кладёт `confidence` в segments → это нормально, не баг.
2. Комментарий добавлен в CRITERIA.md C3.
3. GPU не нужен: model.encode на CPU детерминирован побайтово.
