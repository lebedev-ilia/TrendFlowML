## Описание компонента

`TranscriptAggregatorExtractor` — аггрегатор чанковых эмбеддингов транскрипта поверх вывода `TranscriptChunkEmbedder`. Не запускает модель (только dp_models resolve для metadata). Выполняет mean/max pooling c position-decay weighting + L2-нормализацию → записывает .npy артефакты.

**Вход:** `doc.tp_artifacts.transcripts[source].chunk_embeddings_relpath` → np.ndarray (N, 1024)  
**Выход:** `transcript_{source}_agg_{mean,max}.npy` (1024,) + `features_flat` 19 ключей
