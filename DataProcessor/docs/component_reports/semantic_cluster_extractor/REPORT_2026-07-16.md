## 5. Инфра / оптимизация

- **Под не нужен** — компонент CPU-only, numpy matmul. Весь анализ проведён локально.
- **FAISS optional** — корректный fallback на numpy_cosine при отсутствии FAISS.
- **n_clusters=32** — при таком малом числе FAISS не даёт ощутимого ускорения.
- **Производительность**: compute_ms ≈ 0.1–5ms — пренебрежимо для 200k видео.
