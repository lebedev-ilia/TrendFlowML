## Описание компонента

Bundle-агрегатор: объединяет выходы asr_extractor + speaker_diarization + pitch_extractor в компактный speech-обзор. CPU-only, нет собственных нейросетей — только numpy-агрегация поверх deps. Вызывается через `run_bundle()` (стандартный `run()` не поддерживается — требует Segmenter window families).

## Баг-фикс

**БАГ (найден и исправлен):** `empty_reason="audio_missing_or_extract_failed"` при silent audio → должно быть `"audio_silent"`.

- Файл: `AudioProcessor/src/extractors/speech_analysis_extractor/main.py`, строка ~434
- Проблема: комментарий в коде утверждал «стандартное значение», но все остальные audio-компоненты (speaker_diarization, emotion_diarization, source_separation) используют `"audio_silent"` для тихого аудио
- Фикс: `"audio_missing_or_extract_failed"` → `"audio_silent"` (C5, унификация с экосистемой)
