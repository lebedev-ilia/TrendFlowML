## Логика / оптимизация / инфра

### Логика
- Компонент вычисляет intra-video diversity/repetition из CLIP-эмбеддингов (cosine similarity матрица N×N).
- Метрики: max_sim_to_other (N,), cos_dist_next (N-1,), diversity_score, repetition_ratio, effective_unique_frames — все корректны.
- Otsu-порог auto-calibrates threshold из распределения max_sim_to_other.

### Исправлен баг (L3 из checklist)
- **Баг:** `_otsu_threshold_and_quality_0_1` при all-identical embeddings → `sigma_b2` all-NaN → `np.nanargmax` бросает `ValueError: All-NaN slice encountered`.
- **Фикс:** добавлен guard `if np.all(np.isnan(sigma_b2)): return 0.97, 0.0` перед `nanargmax`.
- **Тест:** синтетика с `np.tile` (30 одинаковых кадров) — PASS, no crash.

### Оптимизация
- CPU-only numpy — под не нужен. N≤200 (max_frames=200) → N×N матрица ≤ 200² = 40K элементов (~160KB fp32). Стоимость ничтожна.
- В проде: переиспользование CLIP-эмбеддингов из core_clip (один раз для всех downstream).
