# REPORT — lexico_static_features

**Дата:** 2026-07-17  
**Версия:** 1.2.0  
**Под:** не нужен (CPU-only компонент)  
**Авто-штамп:** YES (все U1–U6 PASS)

## 1. Компонент

`LexicalStatsExtractor` — CPU-only TextProcessor компонент. Извлекает 67 плоских лексических фич из title/description/transcript видео (regex/unicode/numpy). Нет нейросети, нет модели, нет GPU.

Выход: срез `tp_lex_*` в `text_processor/text_features.npz` (67 ключей, схема `lexico_static_features_output_v1`).

## 2. Логика / оптимизация / инфра

**Логика:** корректна. Граничные случаи обработаны:
- `require_transcript=True` + нет ASR → `RuntimeError` ✅
- `emoji_policy='required'` + нет emoji-lib → fail-fast при инициализации ✅
- All-caps title → `upper_lower_ratio=N/1` (корректно, validator ≥0) ✅
- Пустой транскрипт → все `tp_lex_transcript_*` = NaN by design ✅

**Оптимизация:** компонент дёшев (чистая арифметика), `compute_ms` ≈ 1–5 мс. Узких мест нет.

**Инфра:** валидатор полный (schema/structure/ranges), 67 ключей, one-hot проверки, range-чеки.

## 6. Различимость (детали)

На 6 уникальных видео:
- **14 полей std>0:** `compute_ms`, `description_chars_*`, `punctuation_entropy`, `special_character_ratio`, `title_avg_word_len`, `title_capital_words_ratio`, `title_chars_*`, `title_len_*`, `title_punctuation_ratio`, `title_stopword_ratio`, `upper_lower_ratio_title`
- **26 полей const=0.0:** конфиг (allow_legacy=0, emoji_count=0, urls=0 и т.д.) — все видео с одним описанием тестового Fetcher
- **15 полей const=1.0:** конфиг-флаги (enabled=1, group_*_enabled=1 и т.д.)
- **1 поле const=16.0:** description_len_words (тестовый Fetcher, одинаковые описания)
- **NaN поля:** transcript_* (нет ASR), emoji_diversity (нет эмодзи), named_entity_density (NER заглушка)

## 7. Рекомендации

- `tp_lex_emoji_diversity` = NaN при `emoji_count=0` — потребителям моделей рекомендуется заменять на 0.0 при `emoji_count=0` или использовать как missing-маску. (Задокументировано в audit_v4.)
- NER (tp_lex_named_entity_density) — заглушка. При добавлении NER-компонента обновить версию до v2.
