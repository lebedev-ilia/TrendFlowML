## Замечания / дефекты
- **bg_ratio ≈ 0.40 by design.** Определён как `mean(mag ≤ percentile40(mag))` (main.py:116-118) → доля пикселей ниже 40-го перцентиля ≈ 0.40 почти константно. Валиден (∈[0,1]), но малоинформативен как фича — для Encoder кандидат на исключение/переосмысление порога. Не блокер.
- **ДЕФЕКТ (batch-путь):** `VisualProcessor/utils/core_optical_flow_batch.py` пишет только motion_norm/dt/preview, НЕ заполняет audit-v3 per-frame фичи → batch-NPZ провалит структурный валидатор. Для валидации использован per-video main.py. **Синхронизировать batch перед прод-масштабом 200k.**
- **Инфра:** Triton-сервер с RAFT ensemble не поднят (launch-скрипта нет); прод-путь — inprocess raft_small или, при наличии Triton, тот же v3-артефакт. cut_detection в проде должен переиспользовать этот RAFT (см. REPORT cut_detection).
