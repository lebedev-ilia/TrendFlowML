## Исправленные баги

1. **validate_clap.py (validate_structure)**: пустой embedding → rc=2 при empty/error.  
   Фикс: `if status == "ok" and emb_present_flag:` перед warning.

2. **CLAPExtractor._load_model(): PyTorch 2.12 несовместимость**  
   `torch.load()` default `weights_only=True` несовместим с laion_clap checkpoint (numpy pickle objects).  
   Фикс: monkey-patch `torch.load` → `weights_only=False` перед вызовом `load_ckpt`.  
   Checkpoint из dp_models — доверенный локальный источник.
