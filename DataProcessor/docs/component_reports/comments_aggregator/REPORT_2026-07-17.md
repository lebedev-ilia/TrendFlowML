- корректно обрабатывает empty-path (нет embeddings → present=0, нет исключения если require=False)

- корректно строит три зеркальных семейства ключей
- timing-поля (`agg_mean_ms`/`agg_median_ms`) ненулевые только при `emit_extra_metrics=True`
