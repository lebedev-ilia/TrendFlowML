## Изменения кода (3 фикса в video_processor.py)

**1. keyframes всегда пустые (ФИКС)**
- До: `seq.get("valence")`, `times_s` → NameError молча глотался → keyframes=[]
- После: `seq_proc.get("valence")`, `times_s_proc` (переменные корректного scope)
- Результат: keyframes_n=1/8/13 на 3 реальных видео

**2. nanargmax all-NaN → ValueError (ФИКС)**
- До: `np.nanargmax(emotion_probs, axis=1) if emotion_probs.size` — только защита от size=0
- После: `valid_rows = ~np.all(np.isnan(emotion_probs), axis=1)`, dominant=-1 на пустых строках

**3. use_amp не конфигурируем (ФИКС)**
- До: `use_amp=True` захардкожено → не детерминирован для golden
- После: `os.environ.get("EMOTION_FACE_USE_AMP", "1")` → golden с `EMOTION_FACE_USE_AMP=0` fp32
